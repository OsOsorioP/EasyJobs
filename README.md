Plataforma de inteligencia para la gestión y análisis de perfiles de candidatos, diseñada para cubrir un flujo completo end-to-end: desde la creación y administración de datos hasta la búsqueda semántica y generación de insights mediante LLMs.

## Objetivo del proyecto

Construir un sistema completamente funcional que permita:

- [x] Gestionar perfiles de candidatos mediante un CRUD completo
- [x] Ingestar datos de forma masiva a través de procesos ETL
- [x] Indexar perfiles de forma semántica en un motor vectorial
- [x] Realizar búsquedas avanzadas desde una interfaz web
- [x] Generar insights automáticos utilizando LLMs
- [x] Integrar un microfrontend especializado dentro de la aplicación principal
